# Exercícios React Native

Repositório com exercícios desenvolvidos em React Native durante os estudos de desenvolvimento mobile.

## Conteúdo

O projeto contém diversos exemplos utilizando componentes e recursos importantes do React Native, como:

- SafeAreaView
- ScrollView
- FlatList
- Picker
- Slider
- Flexbox
- Estilização com StyleSheet
- Componentes interativos
